# Basketball Free Throw Game

Welcome to the Basketball Free Throw Game! This project is a fun and interactive game where players can test their free throw shooting skills. The game features player trajectory drawing, speed and direction control, timing, scoring, and three difficulty levels.

## Features

- **Player Control**: Control the player's position and shooting angle.
- **Trajectory Drawing**: Visualize the ball's trajectory before taking a shot.
- **Speed and Direction Control**: Adjust the speed and direction of the shot.
- **Timing**: Shoot within a time limit to score points.
- **Scoring System**: Keep track of successful shots and display the score.
- **Difficulty Levels**: Choose from three difficulty levels that affect game parameters.

## Getting Started

To get started with the Basketball Free Throw Game, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd basketball-freethrow-game
   ```

2. **Open the Project**:
   Open the `index.html` file in your web browser to start playing the game.

3. **Controls**:
   - Use the arrow keys to move the player.
   - Adjust the shooting angle and power using the mouse or keyboard controls (to be defined in the game).
