# 🧬 孟德爾遺傳法則考驗 (互動實驗室)

這是一個基於 HTML5、CSS3 與原生 JavaScript 開發的互動式遺傳學學習工具。使用者可以透過拖放「小精靈 (Blob Spites)」進行繁殖實驗，觀察性狀如何根據孟德爾的第一遺傳法則發生遺傳。

## 🌟 特色功能

- **自由沙盒模式**：支援在實驗區域內任意放置個體，模擬 Scratch 的直覺操作手感。
- **全方位繁育**：親代 (P) 與子代 (F1, F2...) 之間皆可互相交配，支援回交 (Backcross) 或自交實驗。
- **動態顯隱性隨機化**：每次進入實驗室都會抽選不同的特徵（如體色、眼睛大小等），且**顯隱性關係是隨機決定**的（例如：本局大眼睛是顯性，下局可能是隱性），學生必須透過實驗觀察來推論。
- **親代網格排列**：親代 (P) 增加至 6 隻，並以整齊的 2x3 網格隨機分佈在親代區，避免位置暗示答案。
- **基因型考驗限制**：考驗區僅限初始親代進入，確保教學目標聚焦在對未知基因型的科學推論。
- **標準生物標記**：統一使用 **AA, Aa, aa** 符號呈現基因型，與生物課本完全接軌。
- **平板/觸控友善**：支援跨區域拖曳影像不消失，適合行動裝置教學。

## 🎮 使用說明

1. **生殖**：將任意兩個個體拖入中央的「生殖區」，點擊「開始生殖」產生下一代。
2. **分類**：右側為「基因型考驗區」，將**親代表現型**放入對應的基因型 (**AA, Aa, aa**) 槽中。
3. **驗證**：點擊「驗證」按鈕。答對加分並發光，且個體仍可自由移動參與後續實驗；答錯則會扣分且個體會消失。
4. **目標**：獲得 20 分即可成功通關。
5. **重設與清除**：
    - **重設實驗室** (右上角)：更換性狀組合並重啟全新實驗。
    - **清除畫面** (中央)：僅清除「非親代」的個體，保留初始親代以便重新觀察或雜交。

## 🛠️ 教學參數 (URL Parameters)

為了適應不同程度的學生，本程式支援網址參數調整難度：

- **固定性狀顯隱性 (`?fixed=1`)**：
  - **固定 (fixed=1)**：使用預設的顯隱性關係（例如：體色黃色固定為顯性）。適合剛開始接觸遺傳法則的學生。
  - **全隨機 (預設)**：每局隨機分配顯隱性，適合進階挑戰。

- **豌豆模式 (`?mode=pea`)**：
  - **角色切換**：精靈將換成孟德爾經典的高莖與矮莖豌豆。
  - **等位基因提取**：點擊豌豆可呈現其基因型，並「提取」出兩個等位基因。
  - **精確遺傳**：除了直接讓豌豆交配，也可將單個等位基因拖入生殖區，精確控制下一代的基因組合。
  - **生長動畫**：子代會先以「種子」狀態出生，隨後長成成株。

- **進階多性狀混合模式 (`?complex=1`)**：
  - **背景性狀多樣性**：啟動後會將系統中最多達 10 種的性狀（包含翅膀、碎花斑點、觸角末端形狀等）全部加入背景隨機生成，讓每一次生成的個體都極具多樣性，為進階的育種觀察提供更多視覺變化。預設為關閉（只顯示基礎的單一性狀）。

- **目標育種模式 (`?mode=breed`)**：
  - **任務變更**：移除右側的「基因型考驗區」，改為呈現一個隨機生成的「目標個體」。
  - **育種考驗**：玩家必須利用畫面上提供的 8 隻初始親代，透過推論顯隱性關係，將具有相應等位基因的親代進行交配，直到成功繁殖出與「目標個體」具有**完全相同基因型與表現型**的子代，即可獲得加分並進入下一關。

- **參數組合使用**：例如 `index.html?mode=breed&complex=1` 可以產生極具挑戰性的育種遊戲體驗。

## 📁 資料夾結構

- `index.html`：主程式邏輯結構與 SVG 模板。
- `style.css`：版面配置與動態特效。
- `script.js`：核心遺傳邏輯、隨機性狀系統與觸控拖曳實作。
- `assets/audio/`：存放實驗室音效。

本工具旨在提供一個安全且直覺的環境，讓學生能透過實動操作，深刻理解「顯性」與「隱性」在遺傳過程中的運作原理。
